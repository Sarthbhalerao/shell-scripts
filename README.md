## 📂 Scripts Included

### 🔍 Log Analyzer & Report Generator
**Directory:** `Log_Analyzer_and_Report_Generator/`  
- Analyzes system logs  
- Extracts ERROR / CRITICAL events  
- Generates summarized reports  
- Identifies top recurring issues  

---

### 🖥 System Details Script
**Directory:** `get_system_details/`  
- Fetches system information  
- Displays CPU, memory, uptime  
- Useful for quick diagnostics  

---

### 📦 Package Installer
**Directory:** `package_installer/`  
- Installs required packages automatically  
- Checks if package already exists  
- Avoids redundant installations  

---

### 💾 Server Backup Script
**Directory:** `server_backup/`  
- Creates timestamped backups  
- Compresses data using `.tar.gz`  
- Deletes old backups based on retention policy  

---

### ❤️ Service Health Check
**Directory:** `service_health_check/`  
- Checks status of services (nginx, ssh, etc.)  
- Reports whether service is active or not  
- Useful for monitoring scripts
